# Project: My Astro Blog

## Code Style
- **TypeScript Usage:** Use TypeScript for all code; prefer interfaces over types.
- **React Practices:** Write concise, functional components; avoid class-based components.
- **Styling:** Utilize Tailwind CSS utility classes; leverage responsive design utilities.

## Folder Structure
- **Components:** Store reusable UI components in the `src/components` directory.
- **Pages:** Place page-specific components in the `src/pages` directory.

## Deployment
- **Cloudflare Pages:** Optimize assets for performance, considering Cloudflare's CDN capabilities.
- **Build Process:** Ensure compatibility with Cloudflare Pages' build environment.

## Interaction with AI Agent
- Always ask for approval before pushing any code change.
